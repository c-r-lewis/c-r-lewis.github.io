Création d'un site web sécurisé

  • Dates du projet : Octobre 2023 - Décembre 2023
  • Durée du projet : 1 mois et demi
  • Contraintes de travail : Équipe de 3 personnes
Le projet Make Believe Food est un site web principalement développé en PHP, ce qui lui permet d'être connecté à une base de données pour recevoir et stocker des informations liées au commerce du site. L'objectif de ce projet était de renforcer mes compétences en développement web en passant du développement de sites web statiques au développement d'un site web dynamique.
Relation entre les réalisations personnelles et le programme du BUT Informatique
AC - A2 4.02 : Assurer la confidentialité des données

Niveau de maîtrise : acquis

J'ai sécurisé les données sensibles dans la base de données en les cryptant, notamment les mots de passe, et j'ai protégé le code contre les attaques par injection SQL.

AC - A2 4.03 : Organiser la restitution de données à travers la programmation et la visualisation

Niveau de maîtrise : acquis

J'ai conçu une interface web pour que les utilisateurs puissent retrouver et visualiser facilement leurs données, comme leur historique de commandes et leurs informations utilisateur (dans la rubrique des paramètres).

AC - A2 4.04 : Manipuler des données hétérogènes

Niveau de maîtrise : acquis

Le projet impliquait la manipulation d'une grande variété de données : informations sur les produits à vendre, données utilisateur, données de vente, et données destinées à l'administration du site, entre autres.
